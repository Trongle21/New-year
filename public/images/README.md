# Thư mục ảnh

Đặt các ảnh của bạn vào thư mục này.

## Hướng dẫn:

1. Đặt 5 ảnh của bạn vào thư mục `public/images/`
2. Đặt tên ảnh theo thứ tự: `slide1.jpg`, `slide2.jpg`, `slide3.jpg`, `slide4.jpg`, `slide5.jpg`
3. Hoặc bạn có thể đổi tên và cập nhật đường dẫn trong file `src/data/slides.js`

## Định dạng ảnh được hỗ trợ:
- JPG / JPEG
- PNG
- WebP

## Kích thước ảnh đề xuất:
- Tỷ lệ: 16:9 hoặc 4:3
- Độ phân giải: Tối thiểu 800x600px, khuyến nghị 1920x1080px

## Cập nhật nội dung slides:

Mở file `src/data/slides.js` và chỉnh sửa:
- `image`: Đường dẫn đến ảnh (ví dụ: `/images/slide1.jpg`)
- `title`: Tiêu đề của slide
- `description`: Mô tả/nội dung của slide
- `year`: Năm (mặc định là 2025)
